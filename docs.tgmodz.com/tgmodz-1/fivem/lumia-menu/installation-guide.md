# Installation Guide

## 🔑How do you redeem your Key?

{% stepper %}
{% step %}
Create an account at [redEngine Website](https://redengine.eu/signup).
{% endstep %}

{% step %}
Once on the account dashboard you can click "Redeem" on the left to activate your license.
{% endstep %}

{% step %}
{% hint style="danger" %}
Before downloading the software, you will need to disable your anti-virus. This is due to the nature of the software and how it interacts with your game. All products sold by TGModz are completely safe and do not contain any form of malicious code.
{% endhint %}
{% endstep %}

{% step %}
Next click on "Download" on the left and install the redEngine loader. Keep in mind you might need to disable your antivirus software.
{% endstep %}

{% step %}
Run the loader and log in with your redEngine credentials.
{% endstep %}

{% step %}
Load into FiveM and press Inject.
{% endstep %}
{% endstepper %}

{% hint style="danger" %}
NOTE: redENGINE Lua Executor is REQUIRED for Lumia Lua Menu
{% endhint %}
