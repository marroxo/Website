# Installation Guide

## 🔑How do you redeem your Key?

{% stepper %}
{% step %}
Visit the [Ambani website](https://ambani.dev/dashboard).
{% endstep %}

{% step %}
Register your account and on the top of page click Sign Up.
{% endstep %}

{% step %}
Click Your profile top right
{% endstep %}

{% step %}
Click ”**add new licence key**” enter the key provided by TGMODZ&#x20;
{% endstep %}

{% step %}
{% hint style="danger" %}
Before downloading the software, you will need to disable your anti-virus. This is due to the nature of the software and how it interacts with your game. All products sold by TGModz are completely safe and do not contain any form of malicious code.
{% endhint %}
{% endstep %}

{% step %}
You can now download the software above the redeem tab.
{% endstep %}

{% step %}
Before running the loader, make sure the following components are installed.

* [Microsoft Visual C++ Redistributable](https://aka.ms/vs/16/release/vc_redist.x64.exe)
* [DirectX](https://www.microsoft.com/en-us/download/details.aspx?id=35)
  {% endstep %}

{% step %}
Follow any further instructions on screen.
{% endstep %}
{% endstepper %}
