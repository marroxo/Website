# Installation Guide

## 🔑How do you redeem your Key?

{% stepper %}
{% step %}
First go to [Cherax Website](https://cherax.menu/register) and register an account if you haven't already.
{% endstep %}

{% step %}
Once you register your account, a confirmation link will be sent to your email address.
{% endstep %}

{% step %}
Once you confirm your email address, head to the **Redeem Key** Section.
{% endstep %}

{% step %}
Enter your license key, which can be found on the purchase receipt. Then click the **Redeem** button.
{% endstep %}

{% step %}
{% hint style="danger" %}
Before downloading the software, you will need to disable your anti-virus. This is due to the nature of the software and how it interacts with your game. All products sold by TGModz are completely safe and do not contain any form of malicious code.
{% endhint %}
{% endstep %}

{% step %}
Now go to the **Loader Section** to download the loader.
{% endstep %}

{% step %}
Click on download and download the .zip file.
{% endstep %}

{% step %}
Run the loader and login with your Cherax account credentials.
{% endstep %}

{% step %}
Select your game version EE: Enhanced | LE: Legacy, and the game platform.
{% endstep %}

{% step %}
Turn off BattleEye.
{% endstep %}

{% step %}
Click on start, Cherax will open the game along with the menu.
{% endstep %}

{% step %}
Enjoy!
{% endstep %}
{% endstepper %}
