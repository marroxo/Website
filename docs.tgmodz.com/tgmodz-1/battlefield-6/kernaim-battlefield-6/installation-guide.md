# Installation Guide

## 🔑How do you redeem your Key?

### Preparation

{% stepper %}
{% step %}
Download and install Microsoft Visual C++ from [HERE](https://aka.ms/vs/17/release/vc_redist.x64.exe).
{% endstep %}

{% step %}
Uninstall any anti-virus, **deactivation is not enough**.
{% endstep %}

{% step %}
Disable [Real Time Protection and Tamper Protection](https://youtu.be/o0OTed9c_GQ?si=xZXJNXagbYFCxL0m\&t=27) in Windows Defender. You can find help to disable Windows Defender [HERE](https://ash-carpenter-638.notion.site/How-to-disable-Windows-Defender-guide-13678f2021f68054b273dfc260ff6a70).
{% endstep %}

{% step %}
Uninstall **FaceIT** or **Vanguard** background anti-cheats and close any game or launcher.
{% endstep %}
{% endstepper %}
