# Installation Guide

## 🔑How do you redeem your Key?

{% stepper %}
{% step %}
Head to the [Midnight website](https://midnight.im/).
{% endstep %}

{% step %}
Click "Register" in the top right corner and create an account using a username, email, and password.
{% endstep %}

{% step %}
Once registered, log in to your new account and go to the "[User Control Panel](https://midnight.im/usercp/)."
{% endstep %}

{% step %}
Click "Activate Key" and enter your license key.
{% endstep %}

{% step %}
{% hint style="danger" %}
Before downloading the software, you will need to disable your antivirus. This is due to the nature of the software and how it interacts with your game. All products sold by TGModz are completely safe and do not contain any malicious code.
{% endhint %}
{% endstep %}

{% step %}
Go to the "Download Loader" tab and download the Midnight software.
{% endstep %}

{% step %}
Launch the Midnight software as an administrator and log in using your website credentials.
{% endstep %}

{% step %}
Select the relevant game and follow any prompted instructions.
{% endstep %}
{% endstepper %}
