# Installation Guide

## 🔑How do you redeem your Key?

{% stepper %}
{% step %}
Head to the Memesense website here (<https://memesense.gg/auth/register>) and create an account.
{% endstep %}

{% step %}
After registration redeem your key here (<https://memesense.gg/account/activation-keys>) and click on activate key.
{% endstep %}

{% step %}
{% hint style="danger" %}
Before downloading the software, you will need to disable your anti-virus. This is due to the nature of the software and how it interacts with your game. All products sold by TGModz are completely safe and do not contain any form of malicious code.
{% endhint %}
{% endstep %}

{% step %}
Download the launcher from here (<https://memesense.gg/account/download-launcher>)
{% endstep %}

{% step %}
After downloading, run the loader and login with your Memsense credentials.
{% endstep %}

{% step %}
Click on load, the launcher will close automatically and launch the game.
{% endstep %}

{% step %}
Insert key to open and close the menu, use mouse to navigate (click UI)
{% endstep %}
{% endstepper %}
