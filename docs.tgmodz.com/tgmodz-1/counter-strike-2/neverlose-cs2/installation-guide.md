# Installation Guide

## 🔑How do you redeem your Key?

{% stepper %}
{% step %}
Visit the [Neverlose.cc](https://neverlose.cc/) website and register an account. (If you already have one, skip to step 2)
{% endstep %}

{% step %}
Visit the [website ](https://neverlose.tgmodz.com)for redeeming your key
{% endstep %}

{% step %}
Enter your license key and your Neverlose username, then click the "Submit" button and watch your subscription activate instantly.
{% endstep %}
{% endstepper %}

If you receive an error that the NLE balance is too low, then you would need to wait until the balance has been restocked. This usually takes a couple of hours.
