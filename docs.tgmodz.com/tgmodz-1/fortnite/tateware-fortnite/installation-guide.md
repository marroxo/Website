# Installation Guide

## 🔑How do you redeem your Key?

### Pre-Installation Steps

{% stepper %}
{% step %}
Uninstall Antivirus / Anti Malware: Make sure to uninstall (not just "disable") any antivirus or anti-malware software.
{% endstep %}

{% step %}
{% hint style="danger" %}
Before downloading the software, you will need to disable your anti-virus. This is due to the nature of the software and how it interacts with your game. All products sold by TGModz are completely safe and do not contain any form of malicious code.
{% endhint %}

For Windows Defender, use[ Defender Control](https://www.sordum.org/?s=Defender+Control) (run as administrator)
{% endstep %}
{% endstepper %}

### Getting Started

{% stepper %}
{% step %}
Visit Your email to obtain your key.
{% endstep %}

{% step %}
Go to[ UDPCloud](https://udpcloud.com) to register your key.
{% endstep %}

{% step %}
After registration, you will be directed to the main dashboard.
{% endstep %}

{% step %}
Click on Download to get the client.
{% endstep %}
{% endstepper %}

### Running the Loader

{% stepper %}
{% step %}
Launch Rivatuner & Run the Loader:

Open Rivatuner & Launch the loader and log in with the credentials you used to register your license.
{% endstep %}

{% step %}
The loader may close the first time after logging in. Simply reopen it and log in again.
{% endstep %}

{% step %}
Once logged in again, you will be prompted to Press F2 in the lobby -> **ENSURE DISCORD OVERLAY POPS UP BEFORE PRESSING F2**
{% endstep %}

{% step %}
Follow these instructions carefully to ensure a smooth setup and enjoy your game without any interruptions!
{% endstep %}

{% step %}
Enjoy Your Game!
{% endstep %}
