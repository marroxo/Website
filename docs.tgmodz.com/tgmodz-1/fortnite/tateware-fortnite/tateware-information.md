# Tateware Information

## What is Tateware Fortnite?

Tateware offers advanced in-game enhancement tools with precision aiming support, visual overlays, and customizable performance features — built for a smoother and more accessible gameplay experience.

### Compatibility

* Windows 10–11 (1803 - 24H2) — All Windows versions are currently working and up to date.
* Compatible with all major CPU models

### Feature List

* 🎯 **Aim Assistance Features**

  * Customizable Aim Key
  * Trigger Assist Key
  * Bone Target Selector
  * Visibility Check
  * Adjustable Field of View (FOV)
  * Aim Smoothing Settings
  * Crosshair Customization

* 🌈 **Visual Overlay Tools**

  * Configurable Crosshair
  * Dynamic FOV Indicator
  * Aim Lines
  * Player Boxes (with visibility filters)
  * Snaplines (configurable by state)
  * Skeleton Overlay (visibility-based)
  * Distance Display
  * Platform Indicator
  * Weapon Name Display
  * Player Name Display
  * On-Screen Radar System

* 🔫 **Trigger System**

  * Trigger Activation with Custom Key
  * Reaction-based Targeting

* **🛠️ Customization**

  * Adjustable Line Thickness (Skeleton, Snaplines, Boxes)
  * Rounded Box Styling Options
  * Color Picker for All Visual Elements

* 💥 **Advanced Features**

  * Precision Hit Enhancer
  * Enhanced Lighting Mode
  * Impact Boost Mode
  * Replay Sync Assist

* 📡 **Radar Customization**

  * Adjustable Radar Radius
  * Positioning Controls (X/Y)
  * Resizable Radar Window

* 🔧 **Additional Settings**
  * Enable V-Sync
  * Show Real-Time FPS
  * Filter Non-Player Entities (Bots, NPCs, Teammates, Downed Players)
