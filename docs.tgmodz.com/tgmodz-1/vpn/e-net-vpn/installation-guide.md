# Installation Guide

## 🔑How do you redeem your Key?

{% stepper %}
{% step %}
Visit [E-NET Website](https://e-net.cloud/) and create your account.
{% endstep %}

{% step %}
Navigate to dashboard.
{% endstep %}

{% step %}
Click on redeem, copy and paste your license there and click again on Redeem.
{% endstep %}

{% step %}
Navigate to downloads and select Windows App.
{% endstep %}

{% step %}
Download and enjoy secured internet.
{% endstep %}
{% endstepper %}
