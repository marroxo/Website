# Labcore ARC Raiders Information

## What is **Labcore** ARC Raiders?

### Feature List

#### AIMBOT

* Toggle on/off
* Aimbot Toggle Key (F3 customizable F1-F12: Home / End / Insert / Delete)
* Primary Aim Key (customizable)
* Secondary Aim Key (optional with enable/disable toggle)
* Aim Bone Selection
* Aim at Closest Target (center screen vs distance)
* AI Aimbot
  * Toggle on/off
  * Separate AI Aim Key
  * AI Targeting Preference (closest to screen center vs closest by distance)
  * AI Bone Selection (with visual indicator)
* FOV Circle + Color
* FOV Radius (50-500px)
* Prediction Toggle
* Smooth (0-1)
* Randomization (0-5)
* Max Angle Diff (0.1-5)
* Max Distance (0-1000m)
