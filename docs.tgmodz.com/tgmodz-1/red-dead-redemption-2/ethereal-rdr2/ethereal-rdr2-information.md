# Ethereal RDR2 Information

## What is Ethereal RDR2?

**Ethereal RDR2** is a next-generation mod menu designed for Red Dead Redemption 2 players who value control, protection, and creativity. Access powerful features like god mode, teleportation, weapon and player customization, and advanced protections — all within a sleek, optimized interface built for smooth performance in both online and offline modes
